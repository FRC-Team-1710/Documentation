# Overview

Welcome to the official Team 1710 Programming documentation. The goal of this documentation is to preserve knowledge from previous years while providing simple guides to learn the FIRST Robotics Competition programming ecosystem from zero programming knowledge. 

## Where to start

If you are new to the team or this is your first time viewing this documentation, first of all, welcome! From this documentation, there are a few places that might interest you:

- [[From Zero]]
	- This page contains guides to bring you from zero experience to build season ready. Therefore, if you aiming to use this documentation as a guide, start here.
- [[Seasons]]
	- This is essentially an archive of past seasons' documentation and various highlights from past seasons' code. 
- [[Core]]
	- If you are opening this documentation for reference on how our team conducts programming the robot, this is the right place to start. We go over our team's makeup and standards.
- [[Contributing]]
	- Finally, if you want to contribute to this documentation, this is the page to start.
